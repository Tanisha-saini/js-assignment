Explore various console functions:-

Press f12 to see them....

1)console.log ():- It is used to print our message on console

2) console.time():- It starts the timer console.timeEnd():-It calculates the time taken

3)console.table():- It is used to print content in the form of a table

4)console.error():- It prints an error message to the console

5)console.warn():- It prints a warning to console

6)console.clear():- It will clear the console

7)console.assert():- It is used to print a error message whenever an assertion is false
